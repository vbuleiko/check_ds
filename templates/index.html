Как это работает
  1. Выберите один или несколько файлов актов приёмки (.docx)
  2. Нажмите «Разобрать акты» — система извлечёт номер контракта, период, объём км и сумму
  3. Для каждого акта будет показано сравнение с расчётными данными из вкладки Этапы
  4. Нажмите «Подтвердить» рядом с нужным контрактом, чтобы обновить значения и закрыть этап
Исторические этапы (до февраля 2026) уже закрыты и не редактируются. Подтверждение доступно только для расчётных этапов.
